UAV Indoor Navigation Using Sensor Fusion

During my undergraduate research, I focused on developing indoor navigation algorithms for UAVs in GPS-denied environments. Using sensor fusion with data from IMUs, ultrasound sensors, and depth cameras,ac I implemented extended Kalman filters to improve localization accuracy.

Key Achievements:

  • Designed and implemented a dead-reckoning algorithm.
  • Integrated Kalman Filter for sensor fusion.
  • Achieved localization accuracy under 10cm.
